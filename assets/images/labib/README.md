# Labib Images

Place Labib mascot images here.

## Required Image:
- `labib_hero.png` - Main Labib character (recommended: green version from official images)

## Image Requirements:
- Format: PNG (preferably with transparent background)
- Recommended size: 512x512 pixels or similar
- Use one of the Labib superhero images provided

## How to Add:
1. Save one of the Labib images as `labib_hero.png`
2. Place it in this folder
3. Run `flutter pub get`
4. Hot restart the app
